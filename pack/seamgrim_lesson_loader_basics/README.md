# seamgrim_lesson_loader_basics

교과(lesson) 선택 로더 확인용 샘플입니다.

- `c01_load/case.detjson`: 2개 lesson 선택 케이스(DDN 헤더 포함)
- `c01_load/lesson_alpha.ddn`: 헤더 포함 DDN 샘플 1
- `c01_load/lesson_beta.ddn`: 헤더 포함 DDN 샘플 2
- `c01_load/expected_selection_alpha.json`: 선택 상태 스냅샷 기대값 1
- `c01_load/expected_selection_beta.json`: 선택 상태 스냅샷 기대값 2
