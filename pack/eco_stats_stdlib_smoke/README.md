# pack/eco_stats_stdlib_smoke

경제 통계 stdlib(`지니`, `분위수`) 스모크 팩.

- c01: 완전 균등 분포 -> 지니 0
- c02: 비균등 분포 -> 지니 0.266666...
- c03: 분위수 보간(p=0.9) -> 38
- c04: 별칭(`지니계수`, `백분위수`) 동작
- c05: 분위수 범위 오류(p>1) -> E_MATH_DOMAIN
- c06: 최근순위 모드(p=0.9) -> 40
- c07: 분위수 mode 오류 -> E_MATH_DOMAIN
- c08: 분위수 mode 영문값(linear) 거부 -> E_MATH_DOMAIN
- c09: 분위수 mode 한글값(선형보간) 명시 동작
- c10: 분위수 경계 p=0 (선형보간)
- c11: 분위수 경계 p=1 (선형보간)
- c12: 단일 원소 분위수 (선형/최근순위 동일)
- c13: 지니 경계(총합=0) -> 0
