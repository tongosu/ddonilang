# rpg_nurigym_env_smoke_v1

RPG NuriGym 환경 진입 스모크 팩.

## 목적

현재 구현된 `teul-cli nurigym run` 경로로 RPG 던전 최소판을
`nurigym.gridmaze2d` 환경에 실어 episode/dataset을 결정적으로 생성하는지 확인한다.

## 케이스

- `input.json`: 오른쪽 두 칸 이동으로 출구에 도달하는 3x3 던전
