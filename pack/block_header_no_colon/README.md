블록 헤더 무콜론 정본화 pack.
- 입력의 `키워드: {` 표면을 허용한다.
- 정본 출력은 `키워드 {`로 수렴한다.
- 콜론형 블록 헤더 감지 시 `W_BLOCK_HEADER_COLON_DEPRECATED` 경고를 출력한다.
- 커버 범위:
  - `채비: { ... }`
  - `반복: { ... }`
  - `...에 대해: { ... }`
  - `...동안: { ... }`
  - 중첩 블록(`반복: { ... (x) ...에 대해: { ... 동안: { ... } ... } }`)
  - 콜론/무콜론 혼합 입력(정본은 전체 무콜론으로 수렴)
