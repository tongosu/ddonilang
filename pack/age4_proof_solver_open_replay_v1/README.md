# age4_proof_solver_open_replay_v1

`열림.풀이.확인` replay 경계와 `ddn.proof.detjson`의 `solver_open` 요약 봉인이 `중 딱 하나가` 양화와 함께 유지되는지 고정하는 AGE4 회귀 팩.

- family 인덱스:
  - `tests/proof_solver_operation_family/README.md`
  - `python tests/run_proof_solver_operation_family_selftest.py`
- 최상위 proof family:
  - `tests/proof_operation_family/README.md`
  - `python tests/run_proof_operation_family_selftest.py`

- 실행:
  - `python tests/run_pack_golden.py age4_proof_solver_open_replay_v1`
  - `python tests/run_pack_golden.py --update age4_proof_solver_open_replay_v1`
