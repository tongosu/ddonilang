# rpg_quest_smoke_v1

RPG 퀘스트 런타임 스모크 팩.

## 목적

재료 아이템 획득에 따라 퀘스트 진행도가 올라가고, 마지막 재료에서
완료 플래그와 완료 이벤트가 함께 기록되는지 확인한다.

## 케이스

- `input.ddn`: 재료 3개 + 비재료 1개 입력 뒤 `진행=3`, `완료=참`, `완료_이벤트_수=1`
