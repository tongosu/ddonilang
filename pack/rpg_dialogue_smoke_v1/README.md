# rpg_dialogue_smoke_v1

RPG NPC 대화 런타임 스모크 팩.

## 목적

NPC가 대화 시작 알림을 받을 때 방문 횟수를 올리고, 글무늬로 마지막 대사를
결정적으로 만드는지 확인한다.

## 케이스

- `input.ddn`: `용사`와 두 번 대화한 뒤 방문 횟수 `2`, 마지막 대사 `안녕 용사 방문=2`
