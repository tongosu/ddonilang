# connect_endpoint_equality_v1

V1B endpoint inner-sentence `같게` surface evidence.

Scope:
- Accepted: `전지.양극과 전구.왼핀을 (전압은 같게) 잇기`
- Lowered artifact: endpoint path based pack
- Excluded: `흐르게`, `거슬러 흐르게`, `실리게`, whole-object shorthand, `잇기 {}` blocks

This pack does not change V1A function-form `잇기` formula relation semantics.
