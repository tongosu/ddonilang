# rpg_dialogue_template_parse_v1

RPG NPC 대화 기초 파서 + 실행 스모크 팩.

## 목적

NPC 대화 상태(방문 횟수) 추적 DDN이 파서를 통과하고 올바른 값을 출력하는지 확인한다.

## 케이스

- `input.ddn`: 영웅_이름(글)/방문_횟수(수) 채비 + 방문 횟수 증가 + 출력

## 기대 출력

```
1
```
