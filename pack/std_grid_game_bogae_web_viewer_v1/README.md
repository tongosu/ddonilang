# D-PACK: std_grid_game_bogae_web_viewer_v1

## Purpose

Verify that a std grid-game Bogae drawlist can be emitted through the existing `--bogae web --no-open` viewer path.

## Contract

- No new stdlib surface.
- Uses `격자게임보기.보개목록` and `격자게임보기.보개크기`.
- Uses existing `보개_그림판_가로/세로`, `보개_바탕색`, `보개_그림판_목록`, and `보개로 그려`.
