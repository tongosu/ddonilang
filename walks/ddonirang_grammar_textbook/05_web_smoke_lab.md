exposure_level: `전문-실습`  
전제 지식: [04_전문.md](04_전문.md) 완료  
완결 기준: 전문 계층 설명축을 실제 웹/플랫폼 smoke 검증과 혼동하지 않는다.

---

## 5-1. 이 문서의 역할

이 문서는 새 문법을 승격하는 곳이 아니라, 전문 계층 설명 shell 뒤에 오는
**web/runtime smoke 실습 메모 위치**를 고정하는 placeholder다.

- proof / capability / 슬기 / 배움틀의 설명축은 [04_전문.md](04_전문.md)에서 읽는다.
- web smoke lab은 UI/server/adapter baseline 검증을 다루지만 closure claim 을 대신하지 않는다.
- stage A runtime closure가 끝나기 전에는 이 문서를 product completion 근거로 읽지 않는다.

---

## 5-2. 여기서 다루는 것

- representative web smoke runner
- surface regression
- platform adapter / route / warning panel baseline

---

## 5-3. 여기서 다루지 않는 것

- 새 core keyword 승격
- 슬기/배움틀 full runtime closure
- full product completion 선언
