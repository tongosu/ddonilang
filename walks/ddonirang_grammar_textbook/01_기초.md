# 01 기초 — 초급 계층

exposure_level: `기본`  
전제 지식: 없음  
완결 기준: 선언+훅만으로 실행 결과를 확인할 수 있다.

---

## 1-1. 채비 {} — 변수/상수 선언

```ddn
채비 {
  이름: 수 <- 0.
  중력: 수 = 9.8.
}.
```

- `<-` : 변수 (실행 중 바꿀 수 있음)
- `=`  : 상수 (실행 중 불변)
- `: 수` : 타입 표기 (수/정수/글/참거짓)

---

## 1-2. 기본 훅

| 훅 | 실행 시점 |
|---|---|
| `(시작)할때 {}` | 첫 마디 직전 1회 |
| `(매마디)마다 {}` | 매 마디(tick)마다 |
| `(끝날때)할때 {}` | 마지막 마디 직후 1회 |

```ddn
채비 {
  t: 수 <- 0.
  x: 수 <- 0.
}.

(시작)할때 {
  x <- 10.
}.

(매마디)마다 {
  t <- t + 1.
  x <- x - 1.
}.
```

---

## 1-3. 보임 {} — 관찰 출력

```ddn
보임 {
  x축: t.
  값: x.
}.
```

- `x축:` : 가로축 채널 이름
- `값:` : 관찰할 변수

---

## 1-4. 산술 연산

| 연산 | 기호 |
|---|---|
| 더하기 | `+` |
| 빼기 | `-` |
| 곱하기 | `*` |
| 나누기 | `/` |

---

## 1-5. 완전한 초급 예제

```ddn
#기본관찰: 위치

채비 {
  t: 수 <- 0.
  위치: 수 <- 0.
  속도: 수 = 5.
}.

(시작)할때 {
  위치 <- 0.
}.

(매마디)마다 {
  위치 <- 위치 + 속도.
  t <- t + 1.
}.

보임 {
  x축: t.
  값: 위치.
}.
```

## 초급 계층 구문 목록

| 구문 | 분류 | 단계 |
|---|---|---|
| `채비 {}` | 선언 블록 | 1 |
| `= 값` / `<- 값` | 상수/변수 | 1 |
| `: 수` / `: 글` / `: 참거짓` | 타입 | 2 |
| `+`, `-`, `*`, `/` | 산술 | 1 |
| `(시작)할때 {}` | 훅 | 3 |
| `(매마디)마다 {}` | 훅 | 3 |
| `(끝날때)할때 {}` | 훅 | 3 |
| `보임 {}` | 출력 | 1 |
