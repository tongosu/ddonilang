# D-PACK: compound_update_basics

## 목적
- `+<-`/`-<-` 복합 갱신 설탕이 `x <- x ± y`로 정본화되는지 확인한다.
- `+=`/`-=`는 진단으로 거부되는지 확인한다.

## 구성
- `input.ddn`: 기본 동작 샘플
- `tests/README.md`: 수동 실행 가이드

## DoD(최소)
- `teul-cli canon`에서 `+<-`/`-<-`가 `<-` 전개로 출력된다.
- `teul-cli run`에서 기대 출력이 재현된다.
- `+=`/`-=` 입력은 동일 진단으로 거부된다.
