# 다운로드 (한국어)

> 한국어 기준 공개 문서 묶음입니다.

## 배포 위치

- 사용자 공개 바이너리는 GitHub Releases에 둡니다.
- git 저장소에는 사용자용 바이너리를 넣지 않습니다.

## 지원 대상

- Windows x64
- macOS x64/arm64
- Linux x64/arm64

## 권장 파일명

- 'ddonirang-tool-<version>-windows-x64.zip'
- 'ddonirang-tool-<version>-macos-arm64.zip'
- 'ddonirang-tool-<version>-linux-x64.tar.gz'

## 권장 패키지 구조

~~~txt
ddonirang-tool-<version>-<os>-<arch>/
  ddonirang-tool(.exe)
  LICENSE
  NOTICE.txt
  README.txt
~~~

## 체크섬

릴리스에는 SHA256SUMS.txt를 함께 제공합니다. 가능하면 서명도 추가합니다.

## 소스 실행

현재 개발 상태는 소스 빌드와 로컬 셈그림 실행으로 확인합니다. QUICKSTART.md를 보세요.
