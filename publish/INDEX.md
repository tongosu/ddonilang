## 다국어 운영 메모

- 현재 다국어 문서는 README 계열을 기준으로 한다.
- 빠른 시작/개발 구조/문법/다운로드 문서는 기준 문서 하나를 유지하고 README에서 연결한다.
- README 외 문서를 모든 언어로 복제하는 것은 아직 하지 않는다. 실행 명령과 검증 경로가 자주 바뀌기 때문이다.
