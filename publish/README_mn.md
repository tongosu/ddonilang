# 또니랑 (Ddonirang)

> Тоглоом ба AI‑ийн эрин үеийн **солонгос хэл дээр төвлөрсөн** програмчлалын хэл, хэрэгсэл  
> **Детерминизм (дахин гаргалт)** нь “сонголт” биш, **дүрмийн нэг хэсэг** юм.

![또니랑 лого](assets/ddonirang_wordmark.png)

---

## Төслийн тойм

또니랑 нь “бодол шууд тоглоом болдог хэл” гэдэг зорилготой **солонгос төвтэй** хэл, хэрэгслийн төсөл юм.

- **Солонгос хэл дээрх төрөлхийн хэл**: Солонгос хэлний үгсийн дараалал, илэрхийллийг шууд код болгоно. Тодорхой бус байдлыг канонжуулалтаар багасгана.
- **Детерминизм**: Ижил оролт ⇒ хаана ч ижил үр дүн.  
  Алдаа дахин гаргалт, туршилтын харьцуулалт, RL итгэлцэлд “дахин гаргалт” нь үндсэн болно.
- **AI чиг баримжаатай + AI‑aware**: AI код бичсэн ч “итгэлтэй” байлгахын тулд  
  **pack тест + канонжуулалтын golden харьцуулалт + replay/trace**‑ыг хамтад нь төлөвлөнө.
- **Боловсрол/бүтээлд ээлтэй**: Зөвхөн тоглоомоор зогсохгүй, математик/физик/эдийн засгийг “туршилт” болгох хэрэгсэл рүү тэлнэ.
- **말힘누리 (또니랑세상)**: Үг (шившлэг)‑ээр ертөнцийг өөрчилдөг демо/сургалтын ертөнцийг хөгжүүлж байна.

---

## Яагаад хийсэн бэ

Би **бага сургуулиа төгсөхөөс өмнө “дурсагдах бүтээл” үлдээх** хүсэлтэй байсан тул нэг жил бэлтгээд эхэлсэн.

- Python сурах үед: боломж их байсан ч **дүрэм/англи хэлний саад** том байсан.
- Scratch ашиглахад: хөгжилтэй, амархан боловч **хэмжээ томрох тусам хязгаар** мэдрэгддэг.
- Тэгээд сонгосон зорилго:  
  **Солонгосоор уншигдах хэл + тоглоом ба суралцахыг хослуулах ертөнц (누리) + AI‑тай хамтран ажиллах**.

Одоо **эгчтэйгээ “셈그림” (хөдөлгөөнт математикийн зураг)**‑ийг бодитоор хийж,  
“суралцахад тус болох бүтээл” ямар байх ёстойг шалгаж байна.

## Детерминизм нь хөдөлгүүрийн сонголт биш, дүрэм

또니랑 нь **тодорхойгүй тайлбар гарах боломж**‑ыг дүрмийн түвшинд бууруулна.  
Ингэснээр “хүн бичсэн ч, AI бичсэн ч” ижил санаа нэг хэлбэрт ойртдог.

Гол шийдлүүд:

- **Туслах үгийн хил `~` (канон бичлэг)**: Солонгос туслах үгийг код дээр тод харуулна.  
  Жишээ: `문~을`, `주문~으로`, `나~는`
- **Нэг л оноолт `<-`**: Төлөв өөрчлөлтийн тэмдэглэлийг нэг болгоно.
- **Тодорхойлолт vs гүйцэтгэл**: Тодорхойлолт `:움직씨 = { ... }`, гүйцэтгэл `~기 / ~하기`.
- **Нэг холбоос `해서`**: Үйлдлүүдийг холбох хэлбэр нэгтгэгдэнэ.
- **Гэрээ/хамгаалалт**: Зөрчлийг “чимээгүй өнгөрөх” биш, онош/бичлэг (거울) болгон үлдээнэ.

## 30 секундийн жишээ: “Шившлэгээр хаалга нээх”

Шид шиг харагдах маш жижиг жишээ.  
(туслах үгээр холбох + гүйцэтгэлийн төгсгөл + төлөвийн оноолт)

```ddn
바탕.문_상태 <- "닫힘".

(주문:말 ~로) 문~을_열기:움직씨 = {
  { 바탕.문_상태 == "닫힘" }인것 일때 {
    바탕.문_상태 <- "열림".
    (주문) 보여주기.
  } 아니면 {
    "이미 열려있어요." 보여주기.
  }.
}.

("열려라!")~로 문~을_열기 하기.
```

## AI‑тай хамтын ажиллагаа

또니랑 нь “AI код бичиж чадна” гэдгийг үндсэн гэж үздэг.  
Гэхдээ цөмийг детерминист байлгаж, **баталгаажуулах боломжтой урсгал** үүсгэнэ.

- Хүн: зорилго/тайлбар/шалгуурыг тогтооно
- AI: код үүсгэх + санаа/сайжруулалт
- **Pack тест**: жишээ нь golden‑ыг тогтмол давах
- **Канонжуулалт**: хэлбэлзэлтэй бичвэр нэг хэлбэрт буух
- **Replay/trace**: асуудлыг яг дахин гаргах

---

## aiGYM — детерминист AI сургалт/туршилтын орчин

aiGYM нь 또니랑‑ын **детерминизм дээр суурилсан сургалт/туршилтын орчин** юм.  
Ижил оролт, ижил seed‑ээр **заавал ижил үр дүн** гарч, сурах/харьцуулах/дэбаг хийхэд хялбар болно.

- Оролт/лог/replay‑ийг багцалж **дахин гаргах туршилт** болгоно.
- Бодлогын өөрчлөлтийг **pack golden**‑оор харьцуулна.
- Тоглоом/математик/физик/эдийн засгийг **стандарт сургалтын интерфэйс**‑ээр холбоно.

---

## 말힘누리 (또니랑누리/또니랑세상) — үгээр нээгдэх ертөнц

말힘누리 (또니랑누리) нь “үг (шившлэг)‑ээр ертөнцийг өөрчилнө” гэсэн демо/сургалтын ертөнц юм.

- Шившлэг (үг)‑ийг **байгалийн хэлтэй төстэй код** болгон бичнэ
- Ертөнц (누리)‑ийн төлөв **детерминист** байдлаар өөрчлөгдөнө
- Анхан шатны хэрэглэгч “яагаад ажиллаж байна” гэдгийг **replay/trace**‑ээр шалгана

---

## Суралцахад туслах лаборатори (төлөвлөгөө)

또니랑 нь тоглоомоос гадна **суралцахыг туршилт болгох** хэрэгслүүдийг зорьдог.  
Энэ хэсгийг “хэн нэгэнтэй” хамт хийж магадгүй.

### 1) 셈그림 (хөдөлгөөнт математикийн зураг)
- Томьёо/геометр/физикийн хуулийг **“мади таймлайн”**‑аар хөдөлгөнө
- Ижил код → ижил фрейм (дахин гаргалт)
- Математикийн **харагдах** хэрэгсэл

### 2) Физикийн студи (Physics 2D)
- Онол (томьёо) ↔ туршилт (누리) ↔ график (харьцуулалт)
- Чөлөөт уналт/парабол/мөргөлдөөн зэрэг “гар хүрэх” туршилтууд

### 3) Эдийн засгийн лаборатори (Eco‑Lab)
- Бодлого/зах зээлийн туршилтыг слайдераар удирдана
- Зах зээл/урсгал/хамаарлын харагдац + буцаалт (сэргээх)
