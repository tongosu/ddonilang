- 재현 상태는 FAIL 98 / TIMEOUT 1로 Q7 기준선과 일치한다.
- 모든 표 행에 증빙 raw 로그 경로를 기록했다.
- 수리, golden 갱신, 파일 삭제는 수행하지 않았다.
