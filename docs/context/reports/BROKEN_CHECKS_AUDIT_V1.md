# BROKEN_CHECKS_AUDIT_V1

## 범위

- 미션 단계 C 산출물이다.
- `tests/run_*.py` 1080개를 정적 분석했다.
- 분석 대상은 모듈 전역의 필수 문서 경로(`DOC`, `ROADMAP`, `QUEUE`, `PREV`, `REBASE` 등) 중 현재 워킹 트리에 존재하지 않는 `.md` 참조다.
- 이 보고서의 표는 전체 실행 FAIL 목록이 아니라, 누락 `.md` 필수 참조 기반 FAIL 후보(정적 분석) 목록이다.

- 삭제는 실행하지 않았다. 표의 `삭제 심사 후보`는 삭제 실행이 아니라 후속 심사 상신 제안이다.
