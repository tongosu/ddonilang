# SIR 감염 전이(Seed V1)

전체 인구를 `S(감수성)`, `I(감염)`, `R(회복)` 세 집단으로 나눠 시간에 따라 전이되는 과정을 관찰합니다.

## 관찰 안내

- 기본 관찰축: `x=t`, `y=I`
- 함께 보기: `S`, `R`, `new_inf`, `new_rec`
- 슬라이더 실험:
  - `beta`를 키워 감염 확산 정점이 얼마나 빨라지고 커지는지 확인
  - `gamma`를 키워 감염자 감소 속도가 어떻게 빨라지는지 확인
  - `I0`를 바꿔 초기 감염 규모가 전체 곡선에 주는 영향을 비교

## 보개마당 안내

- 도입: 감염자 수(`I`)가 초기에 증가한 뒤 정점을 찍고 감소하는 흐름을 본다.
- 관찰: `beta`, `gamma`, `I0`를 바꿔 정점 높이와 정점 시점을 비교한다.
- 정리: 전파율이 높고 회복률이 낮을수록 유행 정점은 더 크고 늦게 나타난다.

## 실행 구조

- 실행 문법: `채비: {} + (시작)할때 + (매마디)마다 + 보여주기`
