# 재고-가격 피드백 + 수요충격(Seed V2)

재고(`stock`)와 가격(`price`)의 피드백 고리에, 일정 구간 수요 충격을 추가해 조정 과정의 흔들림을 보여줍니다.

## 관찰 안내

- 기본 관찰축: `x=t`, `y=price`
- 함께 보기: `stock`, `demand`, `gap`, `shock_applied`
- 슬라이더 실험:
  - `demand_shock` 크기와 부호를 바꿔 과잉수요/과잉공급 반응을 비교
  - `shock_start`, `shock_end`로 충격 길이를 바꿔 회복 시간 변화를 관찰

## 보개마당 안내

- 도입: 재고가 목표를 벗어나면 가격 조정 신호가 강화된다.
- 관찰: 충격 구간 전후 `stock`과 `price`가 어떻게 오버슈트/언더슈트하는지 본다.
- 정리: 재고 피드백 계수와 충격 지속시간이 안정화 속도를 크게 좌우한다.

## 실행 구조

- 실행 문법: `채비: {} + (시작)할때 + (매마디)마다 + 보여주기`
