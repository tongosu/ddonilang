# 포물선 적분(Seed V1)

포물선 값을 따라가며 구간 적분(사다리꼴 근사) 누적값 변화를 관찰합니다.

## 관찰 안내

- 기본 관찰축: `x=x`, `y=y`
- 함께 보기: `area` 누적값
- 슬라이더 실험:
  - `a`, `b`, `c`를 바꿔 곡선 모양 변화 확인
  - `x_start`, `x_end`를 바꿔 적분 구간 변화 확인
  - `dx`를 줄여 근사 정밀도 변화 확인

## 구조 메모 (호환형)

- 실행 문법은 `#control + (시작)할때 + (매마디)마다 + 보여주기` 경로를 사용합니다.
- `보개마당` 연출은 현재 `#보개마당_*` 메타 주석으로 기록되어 있으며, 정식 블록 전환은 후속 단계에서 적용합니다.
