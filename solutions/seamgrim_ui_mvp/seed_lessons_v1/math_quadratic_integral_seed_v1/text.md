# 포물선 적분(Seed V1)

포물선 값을 따라가며 구간 적분(사다리꼴 근사) 누적값 변화를 관찰합니다.

## 관찰 안내

- 기본 관찰축: `x=x`, `y=y`
- 함께 보기: `area` 누적값
- 슬라이더 실험:
  - `a`, `b`, `c`를 바꿔 곡선 모양 변화 확인
  - `x_start`, `x_end`를 바꿔 적분 구간 변화 확인
  - `dx`를 줄여 근사 정밀도 변화 확인

## 보개마당 안내

- 도입: 포물선 위 점이 이동하면서 면적 누적값(`area`)이 쌓이는 장면을 본다.
- 관찰: `a,b,c`와 적분 구간, `dx`를 바꿔 곡선/누적 면적 변화를 비교한다.
- 정리: `dx`가 작아질수록 사다리꼴 근사 오차가 줄어드는 경향을 확인한다.

## 실행 구조

- 실행 문법: `채비: {} + (시작)할때 + (매마디)마다 + 보여주기`
