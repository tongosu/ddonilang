# 신호등 교통 대기열(Seed V1)

유입률과 신호 주기(초록/빨강)에 따라 대기열(`queue`)이 어떻게 쌓이고 해소되는지 관찰합니다.

## 관찰 안내

- 기본 관찰축: `x=t`, `y=queue`
- 함께 보기: `signal`, `departures`, `delay_index`
- 슬라이더 실험:
  - `green_sec`, `red_sec`를 바꿔 동일 유입률에서 정체 패턴을 비교
  - `in_rate`와 `out_rate` 비율에 따라 대기열 발산/안정 조건을 확인

## 보개마당 안내

- 도입: 신호 주기는 순간 처리율을 주기적으로 바꾼다.
- 관찰: 평균 처리율이 유입률보다 낮아지면 대기열이 누적되는 구간을 확인한다.
- 정리: 주기 최적화는 총 처리량뿐 아니라 지연 변동 폭을 줄이는 데도 중요하다.

## 실행 구조

- 실행 문법: `채비: {} + (시작)할때 + (매마디)마다 + 보여주기`
