# 진자(Seed V1)

중력과 줄 길이에 따라 진자 각도/각속도가 어떻게 변하는지 관찰합니다.

## 관찰 안내

- 기본 관찰축: `x=t`, `y=theta`
- 함께 보기: `omega`
- 슬라이더 실험:
  - `L`을 키워 주기가 길어지는지 확인
  - `g`를 키워 주기가 짧아지는지 확인
  - `theta0`를 바꿔 초기 진폭 변화 확인

## 구조 메모 (호환형)

- 실행 문법은 `#control + (시작)할때 + (매마디)마다 + 보여주기` 경로를 사용합니다.
- `보개마당` 연출은 현재 `#보개마당_*` 메타 주석으로 기록되어 있으며, 정식 블록 전환은 후속 단계에서 적용합니다.
