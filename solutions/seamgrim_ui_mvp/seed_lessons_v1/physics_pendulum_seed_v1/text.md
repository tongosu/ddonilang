# 진자(Seed V1)

중력과 줄 길이에 따라 진자 각도/각속도/에너지가 어떻게 변하는지 관찰합니다.
현재 seed는 `매마디`마다 1스텝씩 전이하는 틱 기반 모델입니다.

## 관찰 안내

- 기본 관찰축: `x=t`, `y=theta`
- 함께 보기: `omega`, `energy`
- 슬라이더 실험:
  - `L`을 키워 주기가 길어지는지 확인
  - `g`를 키워 주기가 짧아지는지 확인
  - `theta0`를 바꿔 초기 진폭 변화 확인
  - `damp`를 키워 진폭 감쇠가 빨라지는지 확인
  - `damp=0`에서 에너지(`energy`)가 거의 일정한지 확인

## 보개마당 안내

- 도입: 초기 각도에서 출발한 추가 좌우로 진동하는 모습을 본다.
- 관찰: `L`, `g`, `damp`를 바꾸며 주기와 감쇠가 어떻게 달라지는지 비교한다.
- 정리: 감쇠가 없으면 에너지가 보존되고, 감쇠가 있으면 시간이 갈수록 진폭이 줄어든다.

## 실행 구조

- 실행 문법: `채비: {} + (시작)할때 + (매마디)마다 + 보임/모양`
- 역할 분리:
  - `채비`: 조절값(슬라이더) 선언
  - `(시작)할때`: 상태 초기화(`t/theta/omega`)만 수행
  - `(매마디)마다`: 상태 갱신 + `보임`/`모양` 출력
- `보여주기`는 소스에서 직접 쓰지 않습니다. `보임`/`모양`을 쓰면 런타임 전처리에서 내부 출력 형식으로만 변환됩니다.
- 상태 전이: 틱 기반 오일러 적분(매 틱 1스텝)
  - `alpha = -(g/L) * sin(theta) - damp * omega`
  - `omega <- omega + alpha * dt`
  - `theta <- theta + omega * dt`

## 모양 작성 규칙

- `모양` 블록 안에는 `선/원/점`만 넣습니다.
- 각 도형은 한 줄 한 문장(`.`)으로 닫습니다.
- 예시:
  - `선(0, 0, bob_x, bob_y, 색="#9ca3af", 굵기=0.02).`
  - `원(bob_x, bob_y, r=0.08, 색="#38bdf8", 선색="#0ea5e9", 굵기=0.02).`
  - `점(0, 0, 크기=0.045, 색="#f59e0b").`
