# 포물선 투사(선형 항력, Seed V1)

초속/발사각에 공기저항(선형 항력)을 더해, 이상 포물선과 다른 감쇠 궤적을 관찰합니다.

## 관찰 안내

- 기본 관찰축: `x=t`, `y=y`
- 함께 보기: `x`, `vy`, `speed2`
- 슬라이더 실험:
  - `drag`를 높여 사거리 감소와 정점 변화량을 확인
  - `angle0`를 바꿔 같은 `v0`에서 궤적 모양 차이를 비교

## 보개마당 안내

- 도입: 항력이 없으면 대칭 포물선, 항력이 있으면 비대칭 궤적이 된다.
- 관찰: `drag` 증가 시 상승/하강 모두 감쇠되어 정점과 사거리가 줄어드는지 본다.
- 정리: 저항 항은 속도 성분을 동시에 깎아 실제 궤적을 짧고 낮게 만든다.

## 실행 구조

- 실행 문법: `채비: {} + (시작)할때 + (매마디)마다 + 보여주기`
