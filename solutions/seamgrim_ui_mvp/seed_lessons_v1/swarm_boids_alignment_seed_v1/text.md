# 군집 정렬 지표모델(Seed V1)

보이드 개별 위치를 직접 적분하지 않고, 정렬 불일치(`mismatch`)와 응집도(`cohesion`) 지표를 시간축으로 관찰합니다.

## 관찰 안내

- 기본 관찰축: `x=t`, `y=cohesion`
- 함께 보기: `mismatch`, `disturb`
- 슬라이더 실험:
  - `align_k`를 키워 정렬 회복 속도 변화를 확인
  - `noise_amp`, `omega`를 바꿔 외란 강도/주파수에 따른 응집도 흔들림을 비교

## 보개마당 안내

- 도입: 군집 정렬은 회복항과 외란항의 경쟁으로 표현할 수 있다.
- 관찰: 정렬계수가 외란을 얼마나 빠르게 상쇄하는지 `cohesion` 곡선으로 본다.
- 정리: 외란이 커도 정렬 이득이 충분하면 군집 응집은 높은 수준으로 유지된다.

## 실행 구조

- 실행 문법: `채비: {} + (시작)할때 + (매마디)마다 + 보여주기`
