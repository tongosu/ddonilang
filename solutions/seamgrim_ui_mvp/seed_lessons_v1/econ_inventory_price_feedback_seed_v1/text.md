# 재고-가격 피드백 시장(Seed V1)

재고가 목표보다 부족하면 가격이 올라가고, 과잉이면 가격이 내려가는 단순 피드백 시장을 관찰합니다.

## 관찰 안내

- 기본 관찰축: `x=t`, `y=price`
- 함께 보기: `stock`, `demand`, `supply`, `gap`
- 슬라이더 실험:
  - `price_k`를 키워 가격 반응 강도를 높이면 진동이 커지는지 확인
  - `stock_k`를 키워 재고 목표 복원력이 커질 때 수렴 속도 비교
  - `demand_slope`, `supply_slope`를 바꿔 균형점 위치 변화를 확인

## 보개마당 안내

- 도입: 목표 재고에서 벗어나면 가격이 자동으로 조정되는 장면을 본다.
- 관찰: `price_k`, `stock_k`를 바꿔 수렴/진동 경향이 어떻게 달라지는지 본다.
- 정리: 피드백 강도가 과하면 빠른 수렴 대신 가격 진동이 커질 수 있다.

## 실행 구조

- 실행 문법: `채비: {} + (시작)할때 + (매마디)마다 + 보여주기`
