# M/M/1 대기열(유체근사, Seed V1)

단일 서버 대기열을 연속시간 근사로 계산해, 도착률(`arrival_rate`)과 처리률(`service_rate`)의 균형을 시각화합니다.

## 관찰 안내

- 기본 관찰축: `x=t`, `y=q`
- 함께 보기: `arrivals`, `served`, `wait_est`, `util`
- 슬라이더 실험:
  - `arrival_rate`를 올려 임계점(혼잡 시작 구간)을 확인
  - `service_rate`를 높여 평균 대기(`wait_est`) 감소 효과를 비교

## 보개마당 안내

- 도입: 대기열 길이는 유입과 처리의 차이 누적으로 결정된다.
- 관찰: `util`이 1에 가까워질수록 `q`와 `wait_est`가 커지는 구간을 본다.
- 정리: 처리률 여유(안전마진)가 작을수록 지연 변동이 크게 증폭된다.

## 실행 구조

- 실행 문법: `채비: {} + (시작)할때 + (매마디)마다 + 보여주기`
