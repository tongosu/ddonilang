# SIR 감염 전이(부스트 구간, Seed V2)

기본 SIR 전이에 `접촉 증가 구간`을 넣어, 같은 모델에서도 정책/환경 변화 시 곡선이 어떻게 달라지는지 보여줍니다.

## 관찰 안내

- 기본 관찰축: `x=t`, `y=I`
- 함께 보기: `S`, `R`, `beta_eff`
- 슬라이더 실험:
  - `contact_boost`를 키워 정점 감염자(`I`)가 얼마나 커지는지 비교
  - `boost_start`, `boost_end`를 바꿔 유행 정점 시점 변화를 관찰

## 보개마당 안내

- 도입: 접촉률이 일정하지 않을 때 전염 곡선이 어떻게 바뀌는지 본다.
- 관찰: 부스트 구간 길이와 강도를 바꿔 `I`의 피크 높이/시점을 비교한다.
- 정리: 같은 평균 전염률이라도 특정 구간 집중 상승은 피크를 크게 만들 수 있다.

## 실행 구조

- 실행 문법: `채비: {} + (시작)할때 + (매마디)마다 + 보여주기`
