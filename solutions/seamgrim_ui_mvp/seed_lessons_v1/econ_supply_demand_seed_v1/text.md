# 수요-공급 가격 조정(Seed V1)

초기 가격 `p0`에서 시작해, 초과수요 `gap = qd - qs`가 가격을 어떻게 밀어 올리거나 내리는지 관찰합니다.

## 관찰 안내

- 기본 관찰축: `x=t`, `y=p`
- 함께 보기: `qd`, `qs`, `gap`
- 슬라이더 실험:
  - `d0`, `d1`를 바꿔 수요곡선 기울기/절편 변화 확인
  - `s0`, `s1`를 바꿔 공급곡선 기울기/절편 변화 확인
  - `k`를 키워 조정 속도(수렴/진동) 차이 확인

## 보개마당 안내

- 도입: 수요와 공급의 차이(gap)가 가격을 밀어 움직이는 장면을 본다.
- 관찰: `d*`, `s*`, `k`를 바꿔 균형점으로 가는 속도와 흔들림을 비교한다.
- 정리: gap이 0에 가까워질수록 가격이 균형 근처에서 안정된다.

## 실행 구조

- 실행 문법: `채비: {} + (시작)할때 + (매마디)마다 + 보여주기`
