# 수요-공급 가격 조정(Seed V1)

초기 가격 `p0`에서 시작해, 초과수요 `gap = qd - qs`가 가격을 어떻게 밀어 올리거나 내리는지 관찰합니다.

## 관찰 안내

- 기본 관찰축: `x=t`, `y=p`
- 함께 보기: `qd`, `qs`, `gap`
- 슬라이더 실험:
  - `d0`, `d1`를 바꿔 수요곡선 기울기/절편 변화 확인
  - `s0`, `s1`를 바꿔 공급곡선 기울기/절편 변화 확인
  - `k`를 키워 조정 속도(수렴/진동) 차이 확인

## 구조 메모 (호환형)

- 실행 문법은 `#control + (시작)할때 + (매마디)마다 + 보여주기` 경로를 사용합니다.
- `보개마당` 연출은 현재 `#보개마당_*` 메타 주석으로 기록되어 있으며, 정식 블록 전환은 후속 단계에서 적용합니다.
