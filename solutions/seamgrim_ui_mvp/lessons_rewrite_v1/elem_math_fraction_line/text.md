# elem_math_fraction_line

math 교과 재작성

## 관찰 안내

- 기본 관찰축: `x=x`, `y=y`
- 함께 보기: (없음)
- 슬라이더 실험:
  - 채비 값을 바꿔 시계열 변화를 비교한다.
  - 시작값과 전이 계수를 조정해 수렴/발산/진동 변화를 확인한다.

## 보개마당 안내

- 도입: math 교과 재작성
- 관찰: 채비 값을 바꾸며 보개 장면과 그래프 변화를 함께 비교한다.
- 정리: 핵심 변수 관계를 한 문장으로 설명한다.

## 실행 구조

- 실행 문법: ``채비: {} + (시작)할때 + (매마디)마다 + 보여주기``
