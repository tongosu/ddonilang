# physics_projectile

초기 속도와 중력 값을 조절해 포물선 궤적(`x,y`) 변화를 관찰한다.

## 관찰 안내

- 기본 관찰축: `x=x`, `y=y`
- 함께 보기: `vy`(세로 속도)는 내부 상태로 감소한다.
- 슬라이더 실험:
  - `vx0`, `vy0`를 바꿔 사거리/최고점 변화를 비교한다.
  - `g`, `dt`를 바꿔 궤적 곡률과 샘플 밀도 변화를 확인한다.

## 보개마당 안내

- 도입: 초기 위치(`x0,y0`)와 속도(`vx0,vy0`)를 정해 발사 조건을 만든다.
- 관찰: 매 틱마다 점이 이동하고 출력행(`x,y`)이 누적되는지 확인한다.
- 정리: `g`가 클수록 `y` 감쇠가 빨라져 최고점이 낮아지는지 정리한다.

## 실행 구조

- 실행 문법: `채비 {} + (시작)할때 + (매마디)마다 + 보임 {}`
