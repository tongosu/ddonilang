# lessons_rewrite_v1/legacy_hold

이 폴더는 대규모 코퍼스 재작성 이전의 참고본을 임시 보관하기 위한 자리다.

- 현재 배치에서는 파일 이동을 강제하지 않는다.
- 대표 교과 재작성은 `physics_motion`, `physics_projectile`, `econ_supply_demand`와 신규 대표군(`physics_pendulum`, `physics_spring`, `econ_demand_curve`, `math_euler`, `math_fibonacci`)에 우선 적용했다.
- 전체 코퍼스 이동/정리는 별도 배치에서 수행한다.
