# 학생 활동지 — P001-02 등가속도운동 v-t 그래프

## 목표
- 등가속도운동에서 속도 v가 시간 t에 선형으로 증가/감소함을 확인한다
- v-t 그래프의 기울기가 가속도 a임을 이해한다

## 준비
`lesson.ddn` 상단의 기본값(살림.*)을 아래처럼 바꾸며 실험한다.

기본값(초기):
- `v0` = 0
- `a` = 1.5
- `t_max` = 10
- `dt` = 0.1

## 해보기
1. a=0일 때 v-t 그래프가 어떤 모양인지 확인해보자.
1. a를 -1로 바꾸고, '감속'을 그래프로 설명해보자.

## 질문(생각해보기)
- a를 음수로 바꾸면 그래프는 어떻게 달라지는가?
- v0를 2로 바꾸면 그래프의 시작점은 어디인가?
