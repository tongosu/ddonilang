# 학생 활동지 — P003-02 감쇠진동 x-t

## 목표
- 감쇠가 있는 진동에서 진폭이 시간이 지날수록 감소함을 그래프로 확인한다
- 감쇠계수 beta가 커지면 감쇠가 빨라짐을 실험으로 확인한다

## 준비
`lesson.ddn` 상단의 기본값(살림.*)을 아래처럼 바꾸며 실험한다.

기본값(초기):
- `x0` = 1
- `v0` = 0
- `omega0` = 2
- `beta` = 0.15
- `t_max` = 20
- `dt` = 0.01

## 해보기
1. beta를 0.05, 0.15, 0.30으로 바꾸어 감쇠 속도를 비교한다.
1. dt를 0.02, 0.01, 0.005로 바꿔서 수치적분의 안정성을 관찰한다(너무 크면 폭주).

## 질문(생각해보기)
- beta를 0.05로 바꾸면 진폭 감소 속도는 어떻게 변하는가?
- dt를 너무 크게 하면 그래프가 왜 '흔들리며' 깨질 수 있을까?
