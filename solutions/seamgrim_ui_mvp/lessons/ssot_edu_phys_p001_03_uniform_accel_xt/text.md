# 학생 활동지 — P001-03 등가속도운동 x-t 그래프

## 목표
- 등가속도운동에서 위치 x가 시간 t에 대해 2차식(포물선) 형태임을 확인한다
- x-t 그래프가 직선이 아닌 이유를 '속도가 변하기 때문'으로 설명한다

## 준비
`lesson.ddn` 상단의 기본값(살림.*)을 아래처럼 바꾸며 실험한다.

기본값(초기):
- `x0` = 0
- `v0` = 0
- `a` = 1.0
- `t_max` = 10
- `dt` = 0.1

## 해보기
1. v0=0, a=1과 v0=3, a=1을 비교해 초기 기울기를 확인해보자.
1. a를 0으로 만들면 x-t가 등속운동과 같아지는지 확인해보자.

## 질문(생각해보기)
- a를 2로 바꾸면 곡선은 더 '휘어지는가'?
- v0를 3으로 바꾸면 그래프의 초기 기울기는 어떻게 달라지는가?
