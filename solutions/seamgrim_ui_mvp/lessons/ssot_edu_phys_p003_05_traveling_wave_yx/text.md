# 학생 활동지 — P003-05 진행파 y-x

## 목표
- 진행파의 공간 파형 y(x)를 그래프로 확인한다
- t0를 바꾸면 파형이 이동(위상 이동)하는 것처럼 보임을 이해한다

## 준비
`lesson.ddn` 상단의 기본값(살림.*)을 아래처럼 바꾸며 실험한다.

기본값(초기):
- `A` = 1
- `lambda` = 4
- `omega` = 2
- `t0` = 0
- `x_min` = 0
- `x_max` = 20
- `dx` = 0.05

## 해보기
1. lambda를 2,4,8로 바꾸어 파장의 길이가 그래프에 어떻게 나타나는지 확인한다.
1. t0를 0, 0.5, 1.0으로 바꾸어 파형이 이동하는지 관찰한다(스냅샷 저장 후 오버레이 추천).

## 질문(생각해보기)
- lambda를 2로 바꾸면 같은 x범위에서 파형은 몇 번 더 반복되는가?
- t0를 1로 바꾸면 그래프는 어느 방향으로 이동한 것처럼 보이는가?
