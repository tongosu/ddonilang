DDN Playground Guide

DDN(또니랑)으로 그래프와 애니메이션을 만들어 보세요.

기본 문법

  • 이름 <- 값. — 변수 대입
  • 이름:타입 = 값. — 타입 지정 대입
  • 매틱:움직씨 = { ... }. — 매 프레임 실행
  • // 한줄 주석
  • 문장 끝에 마침표(.) 필수

보개 그림판 API

보개_그림판_가로, 보개_그림판_세로로 캔버스 크기를 설정합니다.

보개_그림판_목록에 도형 목록을 넣으면 렌더링됩니다.

보개_그림판_목록 <- (
  ("결", "#보개/2D.Rect",
    "x", 10, "y", 20,
    "w", 30, "h", 30,
    "채움색", "#f59e0bff"
  ) 짝맞춤
) 차림.

도형 종류

  • #보개/2D.Rect — 사각형 (x, y, w, h, 채움색)
  • #보개/2D.Circle — 원 (cx, cy, r, 채움색)
  • #보개/2D.Line — 선 (x1, y1, x2, y2, 색)
  • #보개/2D.Text — 글 (x, y, text, 색)

내장 변수

  • 프레임수 — 현재 프레임 번호 (0부터)
  • dt — 프레임 간 시간 (초)

연산자

  • + - * / — 사칙연산
  • % — 나머지(모듈로)
  • < > = != <= >= — 비교
