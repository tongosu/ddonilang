# 표준 격자 가지 (gaji/std_grid)

정수 격자 world/state helper의 나-1 첫실행 surface입니다.

## 공개 API

- `격자.만들기(너비, 높이, 기본값)`
- `격자.너비(격자)`
- `격자.높이(격자)`
- `격자.값(격자, x, y)`
- `격자.바꾼값(격자, x, y, 값)`
- `격자.안인가(격자, x, y)`
- `격자.막혔나(격자, x, y, 막힌값들)`
- `격자.길찾기(격자, 시작x, 시작y, 목표x, 목표y, 막힌값들)`

## 경계

- 격자 canonical state는 정수 좌표와 row-major 칸 목록이다.
- `격자.길찾기`는 BFS 4방향 길찾기이며 이웃 순서는 위, 오른쪽, 아래, 왼쪽으로 고정한다.
- 길찾기 결과는 시작점부터 목표점까지 `(x, y)` 차림의 차림이고, 길이 없으면 빈 차림이다.
- 보개/grid2d/space2d 변환은 view artifact이며 state hash truth를 소유하지 않는다.
- 문서/pack/예제에서는 정본 이름만 사용한다.
