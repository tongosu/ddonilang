# std_grid_game_bogae_live_bridge

Documentation-only skeleton for the std grid-game Bogae finite live web input bridge.

This directory is not loaded by product runtime code. It records the intended usage shape for generated finite live assets and the web input endpoint:

```powershell
teul-cli run pack/std_grid_game_bogae_live_web_assets_v1/input.ddn --madi 4 --madi-hz 1000 --bogae web --bogae-live --sam-live web --bogae-out build/std_grid_game_bogae_live_web_assets_v1 --no-open
```
