# std_grid_game_bogae_web_playback

Documentation-only skeleton for the std grid-game Bogae web playback controls evidence.

This directory is not loaded by product runtime code. It records the intended usage shape for generated static playback assets:

```powershell
teul-cli run pack/std_grid_game_bogae_web_playback_assets_v1/input.ddn --madi 4 --bogae web --bogae-out build/std_grid_game_bogae_web_playback_assets_v1 --no-open
```
