# VS Code DDN Legacy Shell

This folder keeps the older `ddonirang-vscode` extension shell for reference.

The active VS Code shell is the parent folder:

- `tools/vscode-ddn/package.json`
- `tools/vscode-ddn/language-configuration.json`
- `tools/vscode-ddn/README.md`

Do not treat files under this legacy folder as the current extension entrypoint.
Move or delete this folder only after a separate compatibility check confirms
that no packaging or documentation path still references it.
